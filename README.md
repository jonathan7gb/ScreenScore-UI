# ScreenScore-UI

Este é o frontend da API ScreenScore, desenvolvida por [@adryelfelipe](https://github.com/adryelfelipe).

## 🛠️ Stack de Tecnologias

### Linguagem
- [TypeScript](https://www.typescriptlang.org/) — superset tipado do JavaScript

### Framework & Biblioteca Principal
- [React 19](https://react.dev/) — biblioteca para construção de interfaces
- [React DOM](https://react.dev/) — renderização no navegador

### Build & Bundler
- [Vite](https://vitejs.dev/) — ferramenta de build e servidor de desenvolvimento

### Estilização
- [Tailwind CSS v4](https://tailwindcss.com/) — framework CSS utilitário
- [DaisyUI v5](https://daisyui.com/) — biblioteca de componentes para Tailwind CSS

### Ícones
- [Lucide React](https://lucide.dev/) — biblioteca de ícones SVG

### Qualidade de Código
- [ESLint](https://eslint.org/) — linter para JavaScript/TypeScript
- [typescript-eslint](https://typescript-eslint.io/) — regras de ESLint para TypeScript

## API

O repositório da API pode ser encontrado em: [https://github.com/adryelfelipe/ScreenScore-API](https://github.com/adryelfelipe/ScreenScore-API)
